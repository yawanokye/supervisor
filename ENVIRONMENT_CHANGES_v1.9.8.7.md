# Environment Changes - v1.9.8.7

No new environment variable is required.

Keep the existing v1.9.8.6 settings, including:

```env
VPROF_DEVELOPMENTAL_COMMENTS=true
VPROF_COMMENT_DEPTH_FLOOR_ENABLED=true
VPROF_COMMENT_MAX_CHARS=980
VPROF_COMMENT_SIMILARITY_THRESHOLD=0.58
VPROF_STANDARD_RESEARCH_MASTERS_MIN_FINDINGS=18
```

The new section-coverage comments are enabled in the exporter by default because the application must show that every uploaded section was reviewed.
